## 宿舍是上床下桌吗？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
是的，六人间，环境清新，个人空间还不错，买床帘桌帘很有自己的小世界。东校区的床都是新的，不存在床嘎吱嘎吱响的大问题，不过肯定不是一点声音都没有。

:::
## 教室和宿舍有没有空调？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
大部分教室有空调，有些面积小的专业教室就只有风扇。空调一般不开，毕竟遥控器不在学生手里，很热时老师也会开的。宿舍有空调和暖气，空调开关自由，没人管束，不断电。宿舍空调刚开学要交400块的租赁费，有点贵。

:::
## 有独立卫浴吗？没有独立浴室的话，澡堂离宿舍多远？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
东校区没有独立浴室，浴室统一在二楼，共三间，都有隔间，不会尴尬。浴室出门不远处就是吹风机，有白嫖的方法，可以问学长学姐。

:::
## 有早自习、晚自习吗？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
主要看专业。我的专业没有早自习，最早就是早八；有些专业有早七的早自习，详细可问直系学长学姐。大一明面上都安排了晚自习（因为很多同学未成年，适当管束），刚开始严一点，后来就默许不去了，也没人强求。

:::

## 寒暑假放多久，每年小学期有多长？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
寒暑假放得还可以，徐工院放假早是出了名的，考完试就能回。很多学长学姐不着急回去，在学校更自由。基本两个月，暑假两个月，寒假有时也两个月。每学期根据课表19或20周吧，我们学院一般17或18周就考完放假了，比其他专业早。

:::
## 学校允许点外卖吗，取外卖的地方离宿舍楼多远？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
允许点外卖。可以点本校区食堂的，送到宿舍楼下（楼下有架子放外卖）。也可以点其他校区或校外的。学校外的外卖送到校门口（保安室附近有桌子放）。有个学生弄的小程序，点餐能送到宿舍门口，费用贵点。

:::
## 学校交通便利吗，有地铁吗，在市区吗，不在的话进城要多久？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
交通还可以，离地铁站近。骑共享电动车两三块能到地铁站，打车八块左右。坐地铁二号线半个多小时直达市中心彭城广场。校门口也有公交车，但发车时间选择性不如地铁。学校不在市区，但附近有几处小型商场，日常吃饭聚会够用。

:::
## 宿舍楼有洗衣机吗？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
宿舍二楼浴室对面有洗衣间，有三十多个洗衣机。校区新，洗衣机也新（上年换新过），挺干净的。

:::

## 食堂价格贵吗，会吃出异物吗？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
食堂价格正常，和外面差别不大，略便宜，大部分12至16块。不会吃出异物，学校对食堂管理严格，经常检查。食堂里的奶茶店是全国统一价。

:::

::: tabs#faq

## 校园内可以骑电瓶车吗，电池在哪能充电？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
可以骑电动车或自行车。刚开学查得严，需要装学校车牌（不推荐买学长学姐炒高价的二手牌或假牌）。等风头过了再骑就不太抓了。有专门充电区：东校区最东面有几排充电桩，敬中楼南侧也有（少一些）。人多车多，但找找有空位。

:::
## 宿舍限电情况？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
不清楚，没明示限电规则。宿舍六人同时玩电脑或用排插、台灯、充电都没问题。别用大功率电器（被抓到通报批评），也别在宿舍搞火锅（天花板有烟感报警）。

:::
## 通宵自习有去处吗？
::: tabs#faq
@tab 中心校区

@tab 东校区
东校区的自习室不让通宵（十点十一点关门）。宿舍楼内一楼二楼的自习空间没人管，没时间限制。

## 大一能带电脑吗？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
宿舍随时可以带。上课期间看情况：有些课需要用可以带，老师一般不管（别太放肆）；有些课用不到还带就太显眼了。

:::

::: tabs#faq

## 学校的超市怎么样？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
东校区一楼有家大超市，东西基本都有；二楼有家零食牧歌（面积不大）。物价普通，校内不额外贵。但校内水果店比外面贵，建议去附近夜市。

:::
## 学校的收发快递政策怎么样？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
和外面区别不大。取快递要用淘宝扫码出“我的驿站”，复制单号查取件码自取，不花钱。建议避开高峰期（饭后要排队）。

:::
## 学校里面的共享单车数目与种类如何？
::: tabs#faq

@tab 东校区
（24届）那年第一年有共享单车，宣传过。后来慢慢没了（同学不爱惜、有占有欲…）。现在没有了，估计也不会再有了（毁坏严重）。

## 现阶段学校的门禁情况如何？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
东校区进学校不需要刷卡或刷脸，看起来像学生就没人拦。女生宿舍十点门禁，晚一点也让进（别夜不归宿）。男生宿舍普遍没女生严，主要看宿管个人。

:::
## 宿舍晚上查寝吗，封寝吗，晚归能回去吗？
::: tabs#faq
@tab 中心校区
测试1

@tab 东校区
会查寝，但不是每天查，主要看宿管（有的松有的严）。查得很快，够六个人就走，不够会问去哪了。说是十点封寝，但晚一点也能进大门（宿管休息晚），进了大门就有宿管。

:::
